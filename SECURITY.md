# Security

Jenny's Sudoku is a small public personal project. It should not need any API
keys, passwords, or private configuration to run.

Please do not commit:

- `.env` files
- deployment tokens
- private weather or TV API keys
- generated PDFs containing private notes

If you find a security issue, please open a GitHub issue with enough detail to
reproduce it, but do not include secrets or private personal information in the
issue text.
